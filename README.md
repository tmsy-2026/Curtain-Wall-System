# 🏗️ 幕墙工程管理系统 (Curtain Wall Management System)

基于 LocalStorage 的轻量级幕墙工程项目管理系统，支持多项目管理、审批流程、劳务管理等功能。

## 🌐 在线访问

**GitHub Pages 部署地址：**
```
https://tmsy-2026.github.io/Curtain-Wall-System/
```

> ⚠️ 首次访问可能需要等待 1-2 分钟 CDN 缓存生效

---

## 📋 功能模块

| 模块 | 功能 |
|------|------|
| 📊 **仪表盘** | 项目数据总览、快捷入口 |
| 📦 **材料管理** | 材料档案（面材/型材/五金/辅材分类） |
| 📥 **入库管理** | 材料入库记录 |
| 📤 **出库管理** | 材料出库记录 |
| 👷 **劳务管理** | 人员档案、考勤记录、工资管理、进出场登记 |
| 💰 **收支管理** | 收入/支出/备用金管理 |
| 🛒 **采购管理** | 采购申请、合同管理、付款申请 |
| ✅ **审批管理** | 统一审批所有申请流程 |
| 👥 **账号管理** | 用户账号管理（管理员） |
| 💾 **数据备份** | 导入/导出 JSON 数据 |

## 📝 审批流程

| 申请类型 | 流程 | 说明 |
|----------|------|------|
| 采购申请 | 仓管员/劳务员 → **采购员** → **项目经理** | 两级审批 |
| 合同申请 | 采购员 → **项目经理** | 直接终审 |
| 付款申请 | 采购员 → **财务** → **项目经理** | 两级审批，需关联合同 |
| 报销申请 | 任何人 → **财务** → **项目经理** | 两级审批 |
| 备用金借支 | 任何人 → **项目经理** | 直接审批 |

---

## 🛠️ 技术栈

- **前端**：原生 HTML/CSS/JavaScript（单文件架构）
- **存储**：LocalStorage（本地数据持久化）
- **离线**：PWA + Service Worker（支持离线访问）
- **云同步**：腾讯云开发（可选启用）

## 📦 材料分类体系（幕墙专用）

### 面材
铝单板、蜂窝铝板、陶板、瓷板、玻璃、石材、铝塑复合板、不锈钢板等

### 型材
铝合金型材、钢型材、铝方通、钢龙骨、幕墙立柱/横梁等

### 五金
螺栓、螺母、铆钉、驳接爪、执手、门锁、玻璃门夹等

### 辅材
结构胶、耐候胶、保温岩棉、泡沫棒、胶条等
