| 申请类型 | 发起人 | 第一步审批 | 第二步审批 |
|----------|--------|-----------|-----------|
| 采购申请 | 仓管员/劳务员 | 本项目采购员（可部分批准） | 项目经理 |
| 合同申请 | 采购员 | 项目经理 | - |

| 报销申请 | 任何人 | 项目经理 | - |
| 备用金借支 | 任何人 | 项目经理 | - |
